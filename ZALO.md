# Импорт из Zalo

## Скриншоты (рекомендуется)

Подходят **чеки**, **оплата Alipay/WeChat** и **чат Zalo**.

На китайских чеках **часто нет знака ¥** — достаточно строки **合计**, **总计**, **实付** или итоговой суммы.

1. Сфотографируйте чек или скриншот чата  
2. **Zalo** → **Скриншот** → загрузить → **Разобрать**  
3. Проверить → **Добавить выбранные**

**Требуется:** `OPENAI_API_KEY` в Vercel (модель с поддержкой изображений).

- До **5** скриншотов за раз  
- **JPG, PNG, WebP**, макс. **4 МБ** каждый  
- iPhone HEIC: включите «Наиболее совместимые» в настройках камеры  

## Текст (без Vision)

Вкладка **Текст** — вставка скопированных сообщений.  
С `OPENAI_API_KEY` — умный разбор, без ключа — простой парсер.

## Настройка OpenAI

1. [platform.openai.com](https://platform.openai.com) → API keys  
2. Vercel → **Environment Variables** → `OPENAI_API_KEY`  
3. **Redeploy**

Автоматического доступа Zalo к чату **нет** — только скриншоты или копирование текста.
